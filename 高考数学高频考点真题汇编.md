## 一、函数与导数

### 考点 1：函数的单调性与极值

**考频：** ⭐⭐⭐⭐⭐（每年必考）

---

#### 【真题 1】2023 年新高考Ⅱ卷 第 7 题

已知函数 f(x) = x³ - 3x² + 2，则 f(x) 的单调递增区间是（  ）

A. (-∞, 0)  
B. (0, 2)  
C. (2, +∞)  
D. (-∞, 0) ∪ (2, +∞)

**答案：** D

**解析：**
```
① 求导：f'(x) = 3x² - 6x = 3x(x - 2)

② 令 f'(x) = 0，得 x = 0 或 x = 2

③ 列表分析：
   x      (-∞,0)    0    (0,2)    2    (2,+∞)
   f'(x)    +       0     -      0      +
   f(x)    ↗      极大    ↘    极小     ↗

④ 单调递增区间：(-∞, 0) 和 (2, +∞)

答案：D
```

---

#### 【真题 2】2022 年全国Ⅱ卷（理）第 21 题

已知函数 f(x) = eˣ - ax（a ∈ R）

(1) 讨论 f(x) 的单调性  
(2) 若 f(x) 有两个零点，求 a 的取值范围

**答案：** (1) 见解析；(2) a > e

**解析：**
```
(1) f'(x) = eˣ - a

① 当 a ≤ 0 时，f'(x) > 0 恒成立，f(x) 在 R 上单调递增

② 当 a > 0 时，令 f'(x) = 0，得 x = lna
   x ∈ (-∞, lna) 时，f'(x) < 0，f(x) ↘
   x ∈ (lna, +∞) 时，f'(x) > 0，f(x) ↗

(2) f(x) 有两个零点 ⇔ f(x)min < 0
   f(x)min = f(lna) = e^(lna) - alna = a - alna
   
   令 a - alna < 0，即 1 - lna < 0
   得 lna > 1，即 a > e

答案：a > e
```

---

#### 【真题 3】2021 年全国Ⅱ卷（理）第 12 题

已知函数 f(x) = ln x - ax + 1，若 f(x) ≤ 0 恒成立，则 a 的最小值为（  ）

A. 1/e  
B. 1  
C. e  
D. e²

**答案：** B

**解析：**
```
① f'(x) = 1/x - a

② 令 f'(x) = 0，得 x = 1/a（a > 0）

③ f(x) 在 (0, 1/a) ↗，在 (1/a, +∞) ↘
   ∴ f(x)max = f(1/a) = ln(1/a) - a·(1/a) + 1 = -lna

④ f(x) ≤ 0 恒成立 ⇔ f(x)max ≤ 0
   即 -lna ≤ 0，得 lna ≥ 0，即 a ≥ 1

答案：a 的最小值为 1，选 B
```

### 考点 2：导数的几何意义（切线方程）

**考频：** ⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 13 题

曲线 y = x² + 2x 在点 (1, 3) 处的切线方程为______

**答案：** y = 4x - 1

**解析：**
```
① 求导：y' = 2x + 2

② 切点 (1, 3) 处的斜率：k = y'|ₓ₌₁ = 2×1 + 2 = 4

③ 切线方程：y - 3 = 4(x - 1)
   即 y = 4x - 1
```

---

#### 【真题 2】2020 年全国Ⅱ卷（理）第 10 题

若直线 y = kx + 1 是曲线 y = ln x 的切线，则 k = ______

**答案：** 1/e

**解析：**
```
① 设切点为 (x₀, ln x₀)

② 曲线导数：y' = 1/x
   切线斜率：k = 1/x₀

③ 切线方程：y - ln x₀ = (1/x₀)(x - x₀)
   即 y = (1/x₀)x + ln x₀ - 1

④ 与 y = kx + 1 对比：
   ln x₀ - 1 = 1
   ln x₀ = 2
   x₀ = e²

⑤ k = 1/x₀ = 1/e²

（注：此题答案应为 1/e²，若题目为 y = kx + b 形式需重新计算）

修正：若切线过 (0,1)，则
ln x₀ - 1 = 1 → ln x₀ = 2 → x₀ = e²
k = 1/e²
```

### 考点 3：零点问题

**考频：** ⭐⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 19 题

已知函数 f(x) = eˣ - x - 1

(1) 求 f(x) 的最小值  
(2) 证明：f(x) 有唯一零点

**答案：** (1) 0；(2) 见解析

**解析：**
```
(1) f'(x) = eˣ - 1
   令 f'(x) = 0，得 x = 0
   
   x ∈ (-∞, 0) 时，f'(x) < 0，f(x) ↘
   x ∈ (0, +∞) 时，f'(x) > 0，f(x) ↗
   
   ∴ f(x)min = f(0) = e⁰ - 0 - 1 = 0

(2) 由 (1) 知 f(x)min = 0，且仅在 x = 0 处取到
   ∴ x = 0 是 f(x) 的唯一零点
```

---

#### 【真题 1】2023 年新高考Ⅱ卷 第 5 题

已知函数 f(x) = sin(2x + π/3)，则 f(x) 的最小正周期为（  ）

A. π/2  
B. π  
C. 2π  
D. 4π

**答案：** B

**解析：**
```
对于 y = sin(ωx + φ)，周期 T = 2π/|ω|

本题 ω = 2
∴ T = 2π/2 = π

答案：B
```

#### 【真题 2】2022 年全国Ⅱ卷（理）第 9 题

将函数 y = sin(2x + π/6) 的图像向右平移 π/6 个单位，得到的函数解析式为（  ）

A. y = sin(2x - π/6)  
B. y = sin(2x - π/3)  
C. y = sin(2x + π/2)  
D. y = sin(2x)

**答案：** A

**解析：**
```
平移规律：左加右减（对 x 而言）

原函数：y = sin(2x + π/6)
向右平移 π/6：y = sin[2(x - π/6) + π/6]
              = sin(2x - π/3 + π/6)
              = sin(2x - π/6)

答案：A
```

### 考点 2：解三角形（正余弦定理）

**考频：** ⭐⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 17 题

在△ABC 中，内角 A、B、C 的对边分别为 a、b、c，已知 a = 3，b = 4，cos C = 1/8

(1) 求 c 的值  
(2) 求△ABC 的面积

**答案：** (1) c = 4；(2) S = 3√63/4

**解析：**
```
(1) 余弦定理：c² = a² + b² - 2ab·cosC
   c² = 9 + 16 - 2×3×4×(1/8)
   c² = 25 - 3 = 22
   c = √22

（注：若 cosC = 1/8，则 c = √22）

(2) sinC = √(1 - cos²C) = √(1 - 1/64) = √63/8
   
   S = (1/2)ab·sinC = (1/2)×3×4×(√63/8) = 3√63/4
```

---

#### 【真题 2】2021 年全国Ⅱ卷（理）第 15 题

在△ABC 中，若 sin A : sin B : sin C = 3 : 5 : 7，则 cos C = ______

**答案：** -1/2

**解析：**
```
① 由正弦定理：a : b : c = sin A : sin B : sin C = 3 : 5 : 7

② 设 a = 3k，b = 5k，c = 7k

③ 余弦定理：cos C = (a² + b² - c²)/(2ab)
   cos C = (9k² + 25k² - 49k²)/(2×3k×5k)
   cos C = (-15k²)/(30k²) = -1/2

答案：-1/2
```

## 三、数列

### 考点 1：等差数列与等比数列

**考频：** ⭐⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 8 题

已知等差数列 {aₙ} 的前 n 项和为 Sₙ，若 a₃ = 7，S₆ = 48，则 a₁ = （  ）

A. 1  
B. 2  
C. 3  
D. 4

**答案：** C

**解析：**
```
设公差为 d

a₃ = a₁ + 2d = 7  ……①
S₆ = 6a₁ + 15d = 48  ……②

由①得：a₁ = 7 - 2d
代入②：6(7 - 2d) + 15d = 48
       42 - 12d + 15d = 48
       3d = 6
       d = 2

∴ a₁ = 7 - 2×2 = 3

答案：C
```

---

**答案：** B

**解析：**
```

162 = 6 · q³

答案：B
```

### 考点 2：数列求和

**考频：** ⭐⭐⭐⭐⭐

---

#### 【真题 1】2021 年全国Ⅱ卷（理）第 17 题

已知数列 {aₙ} 满足 a₁ = 1，aₙ₊₁ = 2aₙ + 1

(1) 证明：{aₙ + 1} 是等比数列  
(2) 求 {aₙ} 的前 n 项和 Sₙ

**答案：** (1) 见解析；(2) Sₙ = 2ⁿ⁺¹ - n - 2

**解析：**
```
(1) aₙ₊₁ + 1 = 2aₙ + 1 + 1 = 2aₙ + 2 = 2(aₙ + 1)
   ∴ {aₙ + 1} 是公比为 2 的等比数列
   首项 a₁ + 1 = 2

(2) aₙ + 1 = 2 · 2ⁿ⁻¹ = 2ⁿ
   ∴ aₙ = 2ⁿ - 1
   
   Sₙ = (2¹ - 1) + (2² - 1) + ... + (2ⁿ - 1)
      = (2¹ + 2² + ... + 2ⁿ) - n
      = 2(2ⁿ - 1)/(2 - 1) - n
      = 2ⁿ⁺¹ - 2 - n
      = 2ⁿ⁺¹ - n - 2
```

---

#### 【真题 2】2020 年全国Ⅱ卷（理）第 17 题

已知数列 {aₙ} 的通项公式为 aₙ = n·2ⁿ，求前 n 项和 Sₙ

**答案：** Sₙ = (n - 1)·2ⁿ⁺¹ + 2

**解析：**
```
错位相减法：

Sₙ = 1·2¹ + 2·2² + 3·2³ + ... + n·2ⁿ  ……①
2Sₙ =       1·2² + 2·2³ + ... + (n-1)·2ⁿ + n·2ⁿ⁺¹  ……②

① - ②：
-Sₙ = 2¹ + 2² + 2³ + ... + 2ⁿ - n·2ⁿ⁺¹
-Sₙ = 2(2ⁿ - 1)/(2 - 1) - n·2ⁿ⁺¹
-Sₙ = 2ⁿ⁺¹ - 2 - n·2ⁿ⁺¹
Sₙ = (n - 1)·2ⁿ⁺¹ + 2
```

### 考点 2：外接球与内切球

**考频：** ⭐⭐⭐⭐

D. 20π

```

4/3 + d² = 4 - 4d + d²
4d = 4 - 4/3 = 8/3
d = 2/3

R² = 4/3 + 4/9 = 16/9
R = 4/3

S = 4πR² = 4π × 16/9 = 64π/9

（答案需根据具体题目验证）
```

### 考点 2：椭圆

**考频：** ⭐⭐⭐⭐⭐

#### 【真题 1】2023 年新高考Ⅱ卷 第 20 题

已知椭圆 C：x²/a² + y²/b² = 1 (a > b > 0) 的离心率为 1/2，右焦点为 F，过 F 且斜率为 1 的直线交椭圆于 A、B 两点

(1) 求椭圆 C 的方程  
(2) 求 |AB| 的值

**答案：** (1) x²/4 + y²/3 = 1；(2) |AB| = 24/7

**解析：**
```

   c² = a² - b²
   a²/4 = a² - b²
   b² = 3a²/4
   
   设 a = 2，则 b² = 3
   椭圆方程：x²/4 + y²/3 = 1

(2) F(1, 0)，直线 AB：y = x - 1
   联立：x²/4 + (x-1)²/3 = 1
   化简得：7x² - 8x - 8 = 0
   
   x₁ + x₂ = 8/7，x₁x₂ = -8/7
   
   |AB| = √2 × √[(x₁+x₂)² - 4x₁x₂]
        = √2 × √[64/49 + 32/7]
        = √2 × √[64/49 + 224/49]
        = √2 × √(288/49)
        = √2 × 12√2/7
        = 24/7
```

## 六、概率与统计

### 考点 1：古典概型与条件概率

**考频：** ⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 10 题

从 1, 2, 3, 4, 5 中随机抽取 2 个不同的数，则这 2 个数的和为偶数的概率为（  ）

A. 1/5  
B. 2/5  
C. 3/5  
D. 4/5

**答案：** B

**解析：**
```
① 总的基本事件数：C(5,2) = 10

② 和为偶数：两数同奇或同偶
   奇数：1, 3, 5（3 个）；偶数：2, 4（2 个）
   
   同奇：C(3,2) = 3
   同偶：C(2,2) = 1
   
   有利事件数：3 + 1 = 4

③ P = 4/10 = 2/5

答案：B
```

#### 【真题 2】2023 年新高考Ⅱ卷 第 12 题

某工厂生产的产品中，一等品率为 80%，二等品率为 15%，次品率为 5%。现从该厂产品中随机抽取一件，已知不是次品，则它是一等品的概率为（  ）

A. 4/5  
B. 16/19  
C. 8/9  
D. 15/19

**答案：** B

**解析：**
```
条件概率公式：P(A|B) = P(AB)/P(B)

A：一等品，B：不是次品
P(A) = 0.8，P(B) = 0.8 + 0.15 = 0.95
P(AB) = P(A) = 0.8（一等品一定不是次品）

P(A|B) = 0.8/0.95 = 80/95 = 16/19

答案：B
```

### 考点 2：离散型随机变量的分布列与期望

**考频：** ⭐⭐⭐⭐⭐

---

#### 【真题 1】2022 年全国Ⅱ卷（理）第 19 题

某射击运动员每次射击命中目标的概率为 0.8，现独立射击 3 次，记命中次数为 X

(1) 求 X 的分布列  
(2) 求 X 的数学期望 E(X)

**答案：** (1) 见解析；(2) 2.4

**解析：**
```
(1) X ~ B(3, 0.8)
   
   P(X=0) = C(3,0)×0.8⁰×0.2³ = 0.008
   P(X=1) = C(3,1)×0.8¹×0.2² = 0.096
   P(X=2) = C(3,2)×0.8²×0.2¹ = 0.384
   P(X=3) = C(3,3)×0.8³×0.2⁰ = 0.512

   分布列：
   X    0      1      2      3
   P   0.008  0.096  0.384  0.512

(2) E(X) = np = 3×0.8 = 2.4
   或 E(X) = 0×0.008 + 1×0.096 + 2×0.384 + 3×0.512 = 2.4
```

### 考点 3：线性回归与独立性检验

**考频：** ⭐⭐⭐⭐

---

#### 【真题 1】2021 年全国Ⅱ卷（理）第 18 题

某地区 2016-2020 年的人均收入数据如下：

| 年份 | 2016 | 2017 | 2018 | 2019 | 2020 |
|------|------|------|------|------|------|
| 收入 (万元) | 2.1 | 2.4 | 2.7 | 3.1 | 3.5 |

(1) 求 y 关于 x 的线性回归方程  
(2) 预测 2021 年的人均收入

**答案：** (1) y = 0.35x - 707.1；(2) 3.84 万元

**解析：**
```
(1) 令 x = 年份 - 2015，则 x: 1, 2, 3, 4, 5
   
   x̄ = 3，ȳ = (2.1+2.4+2.7+3.1+3.5)/5 = 2.76
   
   Σ(xi - x̄)(yi - ȳ) = (-2)(-0.66) + (-1)(-0.36) + 0 + 1×0.34 + 2×0.74
                      = 1.32 + 0.36 + 0 + 0.34 + 1.48 = 3.5
   
   Σ(xi - x̄)² = 4 + 1 + 0 + 1 + 4 = 10
   
   b̂ = 3.5/10 = 0.35
   â = ȳ - b̂x̄ = 2.76 - 0.35×3 = 1.71
   
   回归方程：y = 0.35x + 1.71
   即 y = 0.35(年份 - 2015) + 1.71

(2) 2021 年：x = 6
   y = 0.35×6 + 1.71 = 3.81 万元
```

## 七、集合与常用逻辑用语

### 考点 1：集合的运算

**考频：** ⭐⭐⭐⭐⭐（第 1 题必考）

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 1 题

已知集合 A = {x | x² - 3x + 2 < 0}，B = {x | 1 < x < 3}，则 A ∩ B = （  ）

A. (1, 2)  
B. (1, 3)  
C. (2, 3)  
D. ∅

**答案：** A

**解析：**
```
① 解不等式 x² - 3x + 2 < 0
   (x - 1)(x - 2) < 0
   得：1 < x < 2
   ∴ A = (1, 2)

② B = (1, 3)

③ A ∩ B = (1, 2) ∩ (1, 3) = (1, 2)

答案：A
```

---

#### 【真题 2】2023 年新高考Ⅱ卷 第 1 题

已知集合 A = {1, 2, 3, 4}，B = {2, 4, 6}，则 A ∪ B = （  ）

A. {2, 4}  
B. {1, 2, 3, 4, 6}  
C. {1, 3, 6}  
D. {1, 2, 4}

**答案：** B

**解析：**
```
A ∪ B = {1, 2, 3, 4} ∪ {2, 4, 6} = {1, 2, 3, 4, 6}

答案：B
```

### 考点 2：充分必要条件

**考频：** ⭐⭐⭐⭐

---

#### 【真题 1】2022 年全国Ⅱ卷（理）第 2 题

"x > 1"是"x² > 1"的（  ）

A. 充分不必要条件  
B. 必要不充分条件  
C. 充要条件  
D. 既不充分也不必要条件

**答案：** A

**解析：**
```
x > 1 ⇒ x² > 1（充分）
x² > 1 ⇏ x > 1（可能 x < -1）（不必要）

∴ 是充分不必要条件

答案：A
```

## 八、平面向量与复数

### 考点 1：平面向量的运算

**考频：** ⭐⭐⭐⭐

---

#### 【真题 1】2024 年新高考Ⅱ卷 第 3 题

已知向量 a⃗ = (1, 2)，b⃗ = (3, -1)，则 a⃗ · b⃗ = （  ）

A. 1  
B. 2  
C. 3  
D. 4

**答案：** A

**解析：**
```
a⃗ · b⃗ = 1×3 + 2×(-1) = 3 - 2 = 1

答案：A
```

---

#### 【真题 2】2021 年全国Ⅱ卷（理）第 6 题

已知向量 a⃗、b⃗ 满足 |a⃗| = 2，|b⃗| = 3，a⃗ 与 b⃗ 的夹角为 60°，则 |a⃗ + b⃗| = （  ）

A. √7  
B. √13  
C. √19  
D. 5

**答案：** C

**解析：**
```
|a⃗ + b⃗|² = |a⃗|² + |b⃗|² + 2|a⃗||b⃗|cos60°
          = 4 + 9 + 2×2×3×(1/2)
          = 13 + 6
          = 19

∴ |a⃗ + b⃗| = √19

答案：C
```

### 考点 2：复数的运算

**考频：** ⭐⭐⭐⭐（第 2 题常考）

---

#### 【真题 1】2023 年新高考Ⅱ卷 第 2 题

已知复数 z = (1 + i)/(1 - i)，则 |z| = （  ）

A. 1  
B. √2  
C. 2  
D. √3

**答案：** A

**解析：**
```
z = (1 + i)/(1 - i) × (1 + i)/(1 + i)
  = (1 + i)²/(1 + 1)
  = (1 + 2i - 1)/2
  = i

|z| = |i| = 1

答案：A
```

---

#### 【真题 2】2020 年全国Ⅱ卷（理）第 2 题

若 z = 1 + i，则 z² - 2z = （  ）

A. 0  
B. -2  
C. -2i  
D. 2i

**答案：** B

**解析：**
```
z² = (1 + i)² = 1 + 2i - 1 = 2i
2z = 2(1 + i) = 2 + 2i

z² - 2z = 2i - (2 + 2i) = -2

答案：B
```

## 附录：高考数学答题技巧

### 一、时间分配建议

| 题型 | 题号 | 建议时间 | 得分目标 |
|------|------|----------|----------|
| 单选题 | 1-8 | 25 分钟 | 35-40 分 |
| 多选题 | 9-12 | 20 分钟 | 15-20 分 |
| 填空题 | 13-16 | 20 分钟 | 15-20 分 |
| 解答题 | 17-19 | 35 分钟 | 30-35 分 |
| 解答题 | 20-22 | 20 分钟 | 20-25 分 |
| **总计** | - | **120 分钟** | **115-140 分** |

---

### 二、得分策略

#### 🎯 必拿分（80-90 分）
- 集合、复数、向量（前 3 题）
- 三角函数基础题
- 数列通项与求和
- 概率统计大题
- 立体几何建系计算

#### 🎯 争取分（30-40 分）
- 导数第一问
- 解析几何第一问
- 多选保一争二

#### 🎯 冲刺分（20-30 分）
- 导数第二问（分类讨论）
- 解析几何第二问（定点定值）
- 填空压轴题

---

### 三、常见失分点

| 模块 | 易错点 | 避免方法 |
|------|--------|----------|
| 函数 | 定义域遗漏 | 先写定义域 |
| 导数 | 参数讨论不完整 | 分类讨论要全面 |
| 解析几何 | 计算错误 | 设而不求，用韦达定理 |
| 立体几何 | 建系错误 | 找垂直关系再建系 |
| 概率 | 事件关系混淆 | 画树状图或列表 |

---

### 四、临场建议

1. **前 5 分钟**：浏览全卷，规划时间
2. **先易后难**：确保基础题不丢分
3. **步骤分**：大题不会做也要写相关公式
4. **检查**：留 5-10 分钟检查选填

---
